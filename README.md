# Beyond True and False: Graded Truth Ratings Conflate Perceived Truth and Certainty
This repository contains the data and code for reproducing all results reported in our paper "Beyond True and False: Graded Truth Ratings Conflate Perceived Truth and Certainty".

All data preparation and analysis is conducted in the scripts [`prep_data.R`](prep_data.R), [`prepare_results.R`](prepare_results.R), and [`results_paper.R`](results_paper.R). All analysis are based on the version of TED at the time of submission (found here: [`data/ted.db`](data/ted.db).

The submitted manuscript deviates from the knit document only in format. To replicate the submission, begin by cloning this repository, installing the relevant packages and executing [`markdown/0_knit_document.R`](markdown/0_knit_document.R).
